🚀 Setup

Up and running in 4 steps

Self-host on any server with Python 3.10+. One webhook, three API keys, and you're live.

1
Clone the repository
Grab the source from GitHub. No build step required — pure Python backend with a minimal dependencies file.
2
Add your API keys
Copy .env.example to .env and fill in your Groq, NVIDIA, Gemini, and GitHub token values.
3
Run the backend
Start the FastAPI server locally or deploy to any cloud. Use ngrok or a reverse-proxy for local webhook development.
4
Configure GitHub Webhook
In your repo Settings → Webhooks, point the URL to your server's /webhook endpoint and select "Pull requests" events.
